# VSCode 101 Course

Introduction to Visual Studio Code and essential features for beginners.

## Overview

This folder contains materials and code examples from the VS Code 101 course, covering the basics of using Visual Studio Code for web and Python development.

## Contents

- **app.js** — JavaScript application example
- **hello.py** — Python "Hello World" example
- **greetings.txt** — Text file example
- **Index.html** — HTML file example
- **Documentation for Visual Studio Code.url** — Link to official VS Code documentation

## Topics Covered

- Setting up VS Code
- Basic editing and navigation
- Built-in terminal usage
- Extensions and customization
- Debugging basics
- Version control integration (Git)
- Multi-language support (Python, JavaScript, etc.)

## Getting Started

1. Open this folder in VS Code: `File > Open Folder`
2. Explore the example files to see different file types
3. Try editing and running the Python, JavaScript, or HTML files
4. Use the integrated terminal: `View > Terminal` or `` Ctrl+` ``

## Quick Tips

- **Open Command Palette:** `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (macOS)
- **Toggle Terminal:** `` Ctrl+` `` (Windows) or `` Cmd+` `` (macOS)
- **Split Editor:** `Ctrl+\` (Windows) or `Cmd+\` (macOS)
- **Format Document:** `Shift+Alt+F` (Windows) or `Shift+Option+F` (macOS)
